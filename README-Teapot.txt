g_buildMode
  0 = No Building (or Deconning)
  1 = Normal Building
  2 = Instant Building
  3 = Heal to Build

g_deconMode
  0 = No Deconning
  1 = Normal Deconning
  2 = Mark Deconstruct

g_nade_damage cvars g_nade_damage and g_nade_range still apply but default to normal behaviour.
g_pathpassword (and !password) for path editing is not recommended but included. Use the 'drawnodes' flag instead.
